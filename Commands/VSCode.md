# Visual Studio Code 명령어 / 단축키

- 자동 정렬: `Ctrl+A` → `Ctrl+K+F`
- 빠른 파일 열기: `Ctrl+P`
- 명령어 보기: `Ctrl+Shift+P`
- 새 터미널 열기: `Ctrl+Shift+\``
- 코드 검색: `Ctrl+Shift+F`
- 컴파일: `Ctrl+Alt+C`
- 실행: `Ctrl+Alt+R`
- 출력: `Ctrl+Alt+N`
- 빌드 생성: `Ctrl+Alt+B`
- 한 줄 주석 토글: `Ctrl+/`
- 블록 주석 토글: `Shift+Alt+A`
- 들여쓰기 앞으로: `Shift+Tab`
- 한 줄 복사: `Shift+Opt+↑/↓`
- Markdown 미리보기: `Cmd+K` → `V`
- Node.js 디버깅: `npm run dev`

| 단축키 | 기능 |
|--------|------|
| `Cmd+L` | 사이드바에 채팅 열기 |
| `Cmd+I` | 선택한 코드 인라인 수정 |
| `Tab` | 자동완성 수락 |
| 코드 선택 후 `Cmd+L` | 선택 코드 컨텍스트로 질문 |
